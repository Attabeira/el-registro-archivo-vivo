# Manifiesto de El Registro – Archivo vivo

Este archivo nace de una convicción sencilla y radical: hay preguntas que no caben por completo ni en la ciencia aislada ni en la espiritualidad encerrada en sí misma.

*El Registro* habita precisamente esa zona de cruce.

Este espacio se abre para prolongar esa exploración. No para clausurarla. No para fijar una doctrina. No para exigir adhesión.

Aquí se reúnen lectores, buscadores, investigadores, creyentes, escépticos y curiosos. Lo que los vincula no es la uniformidad de pensamiento, sino la voluntad de pensar sin reducir la complejidad de lo real.

Creemos que una hipótesis puede ser también una forma de escucha. Que una metáfora puede abrir conocimiento. Que una experiencia no debe ser ridiculizada por no caber de inmediato en el lenguaje dominante. Y que toda apertura intelectual requiere también disciplina crítica.

Este archivo vivo existe para custodiar esa tensión fértil.
