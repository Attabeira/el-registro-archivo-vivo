# El Registro – Archivo vivo

Archivo vivo de diálogo, expansión conceptual y memoria compartida en torno al libro *El Registro: Explorando lo sagrado desde la física cuántica*.

## Bienvenida

Este espacio nace como una prolongación natural del libro *El Registro: Explorando lo sagrado desde la física cuántica*.

Aquí el libro no termina. Aquí respira, se contrasta, se expande y entra en conversación.

Este archivo vivo ha sido concebido para reunir preguntas, referencias, intuiciones, objeciones, experiencias, lecturas paralelas y aportes que dialoguen con los núcleos temáticos de la obra: memoria, conciencia, espiritualidad, física cuántica, simbolismo, experiencia ritual, nodos de sentido y formas posibles de lo sagrado.

No es necesario compartir una misma creencia para participar. Basta con llegar con respeto, curiosidad intelectual y disposición al diálogo.

Este repositorio no busca imponer conclusiones cerradas, sino abrir una cartografía. Cada lector puede aportar una coordenada.

## Propósito

Este espacio ha sido creado para prolongar la vida del libro más allá de sus páginas y abrir una zona de diálogo entre lectores, investigadores, buscadores espirituales, escépticos y curiosos.

No es un tribunal de certezas. Es una cartografía en construcción.
